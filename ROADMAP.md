# Some Perspective -- Project Roadmap

An independent, data-driven research site examining India's political economy (2014-2026).

## Current Release (v2.4.x)

- Grouped navigation with role-based menu structure
- Deep Analysis tab with Sankey flow charts
- Sargam icon integration throughout the UI
- Data refresh incorporating RSF, MOSPI, and PLFS sources
- Role-based navigation for different user contexts

## Next (v2.5)

### New chart sections from unused datasets

- Communal incidents trends
- Opposition incarceration tracker
- Education disruptions timeline
- Sanitation and ODF coverage
- State cess share breakdown

### Other improvements

- Per-chart CSV export buttons for every visualization
- Favicon and social preview image (Open Graph / Twitter Card)

## Future (v3.0)

### Performance and architecture

- Pre-compile Tailwind CSS to remove CDN dependency
- Split the index.html monolith into separate JS and CSS files

### Data expansion

- State-level data breakdowns across all major indicators

### Progressive Web App

- PWA manifest and service worker for offline support

### Continuous integration

- Automated HTML validation on every push
- Link checking for external data sources and references
