1. Methodological Overview

This study employs a mixed-methods approach combining quantitative analysis of official statistics, independent surveys, and international assessments with qualitative analysis of policy documents and institutional changes. The methodology is designed to address the challenge of analyzing an economy where official data reliability is itself a research question.

Core Methodological Principles

  • Triangulation: No single data source is treated as definitive; findings emerge from convergent evidence
  • Transparency: All data sources, transformations, and code are publicly available
  • Conservatism: When data conflict, we report ranges and note uncertainties
  • Replicability: All analyses can be reproduced using provided code and data

5. Triangulation Strategy

5.1 Cross-Validation Matrix

Each finding must be supported by at least three independent sources:

Example: Unemployment Rate Verification

  1. PLFS: 6.1% (2017-18)
  2. CMIE: 7.4% (2017-18)
  3. Labour Bureau: 5.0% (2015-16)
  4. Converged Estimate: 5.5-7.0% range

5.2 Dealing with Suppressed Data

  • Consumption Survey 2017-18: Reconstructed using CMIE household data + state consumption patterns
  • Census 2021: Projected using NFHS-5, voter rolls, and Aadhaar statistics
  • Caste Data: Extrapolated from state surveys (Bihar, Karnataka)

6. Limitations and Caveats

Critical Limitations

  • Census Gap: No census since 2011 limits demographic analysis
  • Consumption Data: 2017-18 survey suppressed; using imputations
  • State Variation: National aggregates mask significant state-level differences
  • Informal Economy: Measurement challenges persist despite methodology improvements
  • COVID Impact: Pandemic effects complicate 2020-2021 comparisons
  • Political Sensitivity: Some data sources may face pressure

8. Replication Guide

8.1 Data Access

Repository: github.com/someperspective/india-economy

Contents:

  • /data/raw/ - Original datasets
  • /data/processed/ - Cleaned data
  • /code/ - Analysis scripts (R, Python, Stata)
  • /docs/ - Codebooks and documentation

8.2 Software Requirements

  • R (v4.3+): Main statistical analysis
  • Python (v3.9+): Data processing and visualization
  • Stata (v17): Robustness checks
  • PostgreSQL: Database management

8.3 Reproduction Steps

  1. Clone repository: git clone https://github.com/someperspective/india-economy
  2. Install dependencies: pip install -r requirements.txt
  3. Run data processing: python process_data.py
  4. Execute analysis: Rscript main_analysis.R
  5. Generate figures: python create_figures.py
